## Getting Started

The Next.js app lives in [`frontend/`](./frontend/).

```bash
cd frontend
npm install
npm run dev
```

From the repository root you can also run `npm install` inside `frontend` once, then use `npm run dev` (delegates to `frontend`).

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Edit pages under `frontend/app/`; the dev server hot-reloads as you save.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
